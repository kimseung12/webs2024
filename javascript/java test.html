01. 퀴즈 이펙트 : 정답 확인하기 유형(1문제)

정답버튼을 누르면 정답이 나오는 퀴즈입니다.

  • 저장하기 : 변수
  • 제어하기 : if문
  • 선택자 : docment.querySelector
  • 메서드 : innerText(), classList.add(), classList.remove()
  • 정답 확인하기
확인하기

문제 정보 입력하기

변수에 문제 정보 입력

{
    const infoDate = "2021년 10월";
    const infoType = "정보처리 기능사";
    const infoNum = "1";
    const infoQuestion = "CPU의 정보처리 속도 단위 중 초당 100만 개의 연산을 수행한다는 의미의 단위는?";
    const infoAnswer = "MIPS";
    const infoDesc = "LIPS는 초당 1개, KIPS는 초당 1000개, MIPS는 초당 1000000의 연산을 수행하는 것을 의미합니다."; 
}

1️⃣ infoDate: "2021년 10월" - 정보가 발행된 날짜를 나타내는 변수입니다.
2️⃣ infoType: "정보처리 기능사" - 정보의 유형이나 범주를 나타내는 변수입니다.
3️⃣ infoNum: "1" - 정보의 번호나 순서를 나타내는 변수입니다.
4️⃣ infoQuestion: "CPU의 정보처리 속도 단위 중 초당 100만 개의 연산을 수행한다는 의미의 단위는?" - 정보에 관한 질문을 나타내는 변수입니다.
5️⃣ infoAnswer: "MIPS" - 질문에 대한 답변을 나타내는 변수입니다.
6️⃣ infoDesc: "LIPS는 초당 1개, KIPS는 초당 1000개, MIPS는 초당 1000000의 연산을 수행하는 것을 의미합니다." - 답변에 대한 설명을 나타내는 변수입니다.

선택자

document.querySelector()

{
    const quizWrap = document.querySelector(".quiz__wrap");
    const quizDate = quizWrap.querySelector(".quiz__date");
    const quizType = quizWrap.querySelector(".quiz__type");
    const quizNum = quizWrap.querySelector(".quiz__question em");
    const quizQuestion = quizWrap.querySelector(".quiz__question span");
    const quizAnswer = quizWrap.querySelector(".quiz__answer");
    const quizDesc = quizWrap.querySelector(".quiz__desc");
    const quizConfirm = quizWrap.querySelector(".quiz__confirm");
    const quizInput = quizWrap.querySelector(".quiz__input input");
    const quizAnimation = quizWrap.querySelector(".quiz__animation");
}

1️⃣ quizWrap: 클래스 이름이 "quiz__wrap"인 요소를 선택합니다. 이 변수는 퀴즈 컨테이너를 나타냅니다.
2️⃣ quizDate: 클래스 이름이 "quiz__date"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈의 날짜를 나타내는 요소를 가리킵니다.
3️⃣ quizType: 클래스 이름이 "quiz__type"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈의 유형을 나타내는 요소를 가리킵니다.
4️⃣ quizNum: 클래스 이름이 "quiz__question em"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈의 번호나 순서를 나타내는 요소를 가리킵니다.
5️⃣ quizQuestion: 클래스 이름이 "quiz__question span"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈의 질문을 나타내는 요소를 가리킵니다.
6️⃣ quizAnswer: 클래스 이름이 "quiz__answer"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈의 정답을 표시하는 요소를 가리킵니다.
7️⃣ quizDesc: 클래스 이름이 "quiz__desc"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈 정답에 대한 설명을 표시하는 요소를 가리킵니다.
8️⃣ quizConfirm: 클래스 이름이 "quiz__confirm"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 사용자가 퀴즈를 확인하는 데 사용할 버튼 또는 요소를 가리킵니다.
9️⃣ quizInput: 클래스 이름이 "quiz__input input"인 입력 요소를 quizWrap 내에서 선택합니다.
이 변수는 사용자가 정답을 입력하는 데 사용하는 입력 상자를 가리킵니다.
🔟 quizAnimation: 클래스 이름이 "quiz__animation"인 요소를 quizWrap 내에서 선택합니다.
이 변수는 퀴즈와 관련된 애니메이션 또는 그래픽 효과를 가리킵니다.

문제 출력 하기

.innerText

{
    quizDate.innerText = infoDate;
    quizType.innerText = infoType;
    quizNum.innerText = infoNum;
    quizQuestion.innerText = infoQuestion;
    quizAnswer.innerText = infoAnswer;
    quizDesc.innerText = infoDesc; 
}

1️⃣ quizDate.innerText = infoDate;: 이 줄은 quizDate라는 HTML 요소에 infoDate 변수의 값을 할당합니다.
즉, infoDate 변수의 내용을 quizDate 요소의 텍스트로 설정합니다.
2️⃣ quizType.innerText = infoType;: 이 줄은 quizType이라는 HTML 요소에 infoType 변수의 값을 할당합니다.
infoType 변수의 내용을 quizType 요소의 텍스트로 설정합니다.
3️⃣ quizNum.innerText = infoNum;: 이 줄은 quizNum이라는 HTML 요소에 infoNum 변수의 값을 할당합니다.
infoNum 변수의 내용을 quizNum 요소의 텍스트로 설정합니다.
4️⃣ quizQuestion.innerText = infoQuestion;: 이 줄은 quizQuestion이라는 HTML 요소에 infoQuestion 변수의 값을 할당합니다.
infoQuestion 변수의 내용을 quizQuestion 요소의 텍스트로 설정합니다.
5️⃣ quizAnswer.innerText = infoAnswer;: 이 줄은 quizAnswer라는 HTML 요소에 infoAnswer 변수의 값을 할당합니다.
infoAnswer 변수의 내용을 quizAnswer 요소의 텍스트로 설정합니다.
6️⃣ quizDesc.innerText = infoDesc;: 이 줄은 quizDesc라는 HTML 요소에 infoDesc 변수의 값을 할당합니다.
infoDesc 변수의 내용을 quizDesc 요소의 텍스트로 설정합니다.

정답 해설 숨기기

.classList.add("none");

{
    quizAnswer.classList.add("none");
    quizDesc.classList.add("none"); 
}

1️⃣quizAnswer.classList.add("none");: 이 줄은 quizAnswer라는 HTML 요소에 "none"이라는 CSS 클래스를 추가합니다. 이렇게 하면 해당 요소에 "none" 클래스가 적용되어 CSS 스타일이 적용됩니다. "none" 클래스가 보통 특정 요소를 숨기거나 숨은 상태로 만들 때 사용됩니다.
2️⃣quizDesc.classList.add("none");: 이 줄은 quizDesc라는 HTML 요소에 "none"이라는 CSS 클래스를 추가합니다. 마찬가지로 해당 요소에 "none" 클래스가 추가되어 숨김 처리됩니다.

정답 확인하기

.addEventListener()

{
    quizConfirm.addEventListener("click", () => {
        const userAnswer = quizInput.value;

        if(infoAnswer == userAnswer){
            quizAnimation.classList.add("like");
        } else {
            quizAnimation.classList.add("dislike");
            quizAnswer.classList.remove("none");
            quizDesc.classList.remove("none");
        }
        quizConfirm.classList.add("none");
        quizInput.classList.add("none");
        quizAnswer.classList.remove("none");
        quizDesc.classList.remove("none");
    });
}

1️⃣ quizConfirm.addEventListener("click", () => { ... });: 이 부분은 "quizConfirm"라는 HTML 요소에 클릭 이벤트 리스너를 추가하는 코드입니다. 사용자가 버튼을 클릭하면 아래의 코드 블록이 실행됩니다.
2️⃣ const userAnswer = quizInput.value;: 사용자가 입력한 답변을 "quizInput" 요소에서 가져와서 "userAnswer" 변수에 저장합니다.
3️⃣ if (infoAnswer == userAnswer) { ... } else { ... }: 사용자의 답변과 정답을 비교하는 조건문을 사용합니다. 만약 사용자의 답변과 정답이 일치하면 첫 번째 블록이 실행되고, 그렇지 않으면 두 번째 블록이 실행됩니다.
4️⃣ quizAnimation.classList.add("like");: 정답이면 "quizAnimation" 요소에 "like" 클래스를 추가하여 어떤 긍정적인 애니메이션 또는 효과를 표시합니다. 이것은 사용자에게 정답을 맞춘 것을 시각적으로 보여주는 데 사용됩니다.
5️⃣ quizAnimation.classList.add("dislike");: 오답일 경우 "quizAnimation" 요소에 "dislike" 클래스를 추가하여 부정적인 효과를 표시합니다.
6️⃣ quizAnswer.classList.remove("none"); 및 quizDesc.classList.remove("none");: 오답 또는 정답일 경우 "quizAnswer"와 "quizDesc" 요소의 "none" 클래스를 제거하여 정답과 그에 대한 설명을 표시합니다.
7️⃣ quizConfirm.classList.add("none"); 및 quizInput.classList.add("none");: 사용자가 정답을 제출한 후 "quizConfirm" 버튼과 "quizInput" 입력 상자를 숨깁니다.
